模块功能概述

作用

1.指定ePWM模块的周期或者频率，时间多久发生 2.管理和其他ePWM模块的时基同步 3.管理和其他ePWM模块的相位关系 4.设定时基计数器的工作方式，count-up, count-down, count-up-and-count-down 5.产生下面的事件

CTR = PRD，时基计数器的值等于指定的周期(TBCTR = TBPRD) CTR = Zero，时基计数器的值等于0

6.配置时基计数器的时钟

相关寄存器

TBCTL Time-Base Control Register

TBSTS Time-Base Status Register

TBPHSHR HRPWM Extension Phase Register

TBPHS Time-Base Phase Register

TBCTR Time-Base Counter Register

TBPRD Time-Base Period Register
