指针数组

array of pointer, 通常用来构造一个字符串的数组，也简称字符串数组。数组的每一个 元素都指向一个字符串，但是在C语言中，一个字符串实质上就是指向其第一个字符的指针 。所以，字符串数据的每个数据项实际上就是一个指向字符串第一个字符的指针

eg. const char *suit[4] = {"Hearts", "Diamonds", "Clubs", "Spades"}; 声明中的char*表示数组suit中的每个元素都是"指向字符的指针"类型

函数指针

一个指向函数的指针(pointer to a function)包含的是一个函数在内存中的地址。指 向函数的指针既可以作为函数实参传递给函数，也可以作为函数返回值从函数返回， 还可以将其存放到数组中和赋值给其他函数指针

void max_min(int a[], int n, int *max, int *min)
{
    int *p ;
    *max = *min = *a ;
    for(p = a + 1; p < a + n; p++)
    {
        if(*p > *max)
        {
            *max = *p ;
        }
        else if(*p < *min)
        {
            *min = *p ;
        }
    }
}
