0 或 s: 查找本 C 符号 1 或 g: 查找本定义 变量的声明处，函数的定义处，非原型声明 2 或 d: 查找本函数调用的函数 3 或 c: 查找调用本函数的函数 4 或 t: 查找本字符串 6 或 e: 查找本 egrep 模式 7 或 f: 查找本文件 8 或 i: 查找包含本文件的文件
