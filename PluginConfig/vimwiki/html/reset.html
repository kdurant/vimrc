git-reset

Reset current HEAD to the specified state

撤销所有暂存区文件, 仅用HEAD指向的目录树重置stage, working directory不受影响 git reset git reset HEAD

彻底回退到某个版本, working directory, stage, repo内容相同 git reset --hard version_name git reset --hard HEAD^

只更改引用的指向, 不改变working directory, stage git reset --soft version_name git commit -m "xxx" 这样可以用来修改注释

从stage中移除指定文件, 但还是在 working directory git reset -- filename git reset HEAD filename
