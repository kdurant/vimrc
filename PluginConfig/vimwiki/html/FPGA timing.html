TNS: total negetive slack, 指定时钟域中所有总负裕量的总和。使用这个值来测量指定 时钟域中失败路径的总和

R ----Rising output F ----Falling output RR ----Rising input, rising output RF ----Rising input, falling output FR ----Falling input, rising output FF ----Falling input, falling output

CELL: cell delay, 对应为一个LE或者别的单元内部延迟，由于内部延迟，所以时间很短。

一般由器件本身决定

IC: Interconnect delay, 对应为LE或者别的单元之间的连接延迟。可以通过时序约束

去约束单元的分布

Minimum Pulse width: 触发器能够识别的最小脉冲宽度

input delay: 表示输入信号相对于主clk的延迟
