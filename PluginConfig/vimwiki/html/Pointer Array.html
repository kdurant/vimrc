array of pointer, 通常用来构造一个字符串的数组，也简称字符串数组。数组的每一个 元素都指向一个字符串，但是在C语言中，一个字符串实质上就是指向其第一个字符的指针 。所以，字符串数据的每个数据项实际上就是一个指向字符串第一个字符的指针

example

int *num[3] ; 声明一个指针数组,该指针数组包含3个数组元素,每个元素都是一个指针变量.
const char *suit[4] = {"Hearts", "Diamonds", "Clubs", "Spades"}; 声明中的char*表示数组suit中的每个元素都是"指向字符的指针"类型
"Hearts" "Diamonds" Clubs" "Spades"
suit[0] suit[1] suit[2] suit[3]
